# 🚗 Car Price Predictor

A Machine Learning project that predicts the selling price of a used car based on various vehicle features such as brand, model, vehicle age, fuel type, transmission type, engine capacity, mileage, and kilometers driven.

---

## 📌 Project Overview

The objective of this project is to build a regression model capable of estimating the selling price of a used car using historical car sales data.

This project demonstrates the complete Machine Learning workflow:

- Data Collection
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Price Prediction

---

## 📂 Dataset

**Dataset Name:** CarDekho Used Cars Dataset

### Features Used

- Brand
- Model
- Vehicle Age
- Kilometers Driven
- Seller Type
- Fuel Type
- Transmission Type
- Mileage
- Engine Capacity
- Maximum Power
- Number of Seats

### Target Variable

- Selling Price
